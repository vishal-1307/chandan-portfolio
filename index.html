Car Parking Management System

This project is a console-based application developed in C++ to manage a car parking facility. It handles the entry and exit of vehicles, calculates parking charges, and keeps track of the number of available parking slots. The system uses fundamental data structures to efficiently store and retrieve vehicle data.
The primary objective was to apply object-oriented programming (OOP) principles and data structure concepts to solve a real-world problem. This project demonstrates my proficiency in C++ and my ability to design logical, efficient systems.
